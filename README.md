## Categorical Variables

- Nominal
  - Nominal variables describe categories that do not have a specific order to them. 
  - Example: [male, female]
- Ordinal
  - Ordinal variables have two are more categories that can be ordered or ranked. 
  - Example: [low, medium, high]
